TransitionRect

Singleton Reference

Singleton used to control fade in and fade out effects.

Inherits from CanvasLayer

Description

This class displays a black rectangle on top of the scene at all times. At the start, it is invisible so as not to cover the screen. Functions can be called to fade the screen in and out by interpolating the alpha of the black rectnggle.

The functions fade_in() and fade_out() are used to implement traditional fade in and fade out effects.

Constants

Type Name Value Brief
float DEFAULT_DURATION 1.0 The default duration of a fade.

Signals

• fade_finished(String type)

Methods

• void fade_in(Dictionary options = {})

Start a fade-in effect.

This function interpolates the alpha value of a black rectangle from one to zero. Certain aspects of the fade, such as fade-time and the initial value, can be controlled via options.

options is a dictionary that contains various parameters for the fade: the key initial_value is used to specify the initial alpha value at the start of the fade, and duration is used to specify how long the fade lasts in seconds.

duration must be a positive number greater than zero in order to be read. If this parameter is not provided or is invalid, it defaults to DEFAULT_DURATION.

The option initial_value defaults to 1.0 unless it is specified by the user. For it to be valid, the initial value must be in the range [0.01, 1.0].

• void fade_out(Dictionary options = {})

Start a fade out effect.

This function causes a fade-out effect by interpolating the alpha value of a black rectangle covering the screen, from zero to one.

options is a dictionary that contains various parameters for the fade: the key initial_value is used to specify the initial alpha value at the start of the fade, and duration is used to specify how long the fade lasts in seconds.

duration must be a positive number greater than zero in order to be read. If this parameter is not provided or is invalid, it defaults to DEFAULT_DURATION.

The option initial_value defaults to zero unless it is specified by the user. For it to be valid, the initial value must be in the range [0.0, 0.99].
