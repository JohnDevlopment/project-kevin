BackgroundMusic

Singleton Reference

Singleton for playing background music.

Inherits from AudioStreamPlayer

Properties

TypeNameDefault
Dictionarysongs

• Dictionary songs — A list of songs each mapped to a name.

This holds a bunch of songs, mapping each name of a file relative to MUSIC_PATH. Each call to play_music() is expects a key from this dictionary.

Constants

Type Name Value Brief
String MUSIC_PATH "res://assets/audio/music" Path to the music folder

Methods

• void fade_out(float duration)

Fades the music out.

Call this function to fade the music over a period of time specified by duration seconds. Once the fade is complete, the music stops.

• void play_music(String song, float position = 0.0)

Play a song from the mapping.

Plays a song from the mapping specified by song. If position is specified and is greater than zero, marks the position in the song (in seconds) to start playback.
