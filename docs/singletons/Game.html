Signals

• changed_game_param(String param, Variant value)

Indicates that a game parameter has changed.

This signal is emitted when certain functions are called. Which function was called is indicated by param.

Param: dialog_mode
Type: bool
Emitted when set_dialog_mode() is called. If true, then dialog mode is active: in this mode, a dialog is in process, so actors should not move or accept input.

Param: tree_paused
Type: bool
Emitted when set_paused() is called.

Methods

• Variant set_dialog_mode(bool enabled)

Set or clear dialog mode.

Depending on whether enabled is true or false, this sets or disables dialog mode. In dialog mode, affected nodes stop working and do not accept input from the user, allowing dialog boxes to work without interruption.

Emits the changed_game_param signal.

• void set_paused(bool paused)

Set the pause status of the scene tree.

Pauses the scene tree if paused is true or unpauses it otherwise.

Emits the changed_game_param signal.
