DebugOverlay

Debug overlay

Inherits from CanvasLayer

Methods

• void add_stat(String stat_name, Object object, String stat_ref, bool is_method)

Adds a stat to the overlay.

Displays information about a stat on the debug overlay. stat_ref is a path to a property or the name of a method inside object. Specify whether it is a method or not with is_method. stat_name is the display name of the stat.

• void remove_stat(String stat_name)

Remove a stat from the overlay.

Removes a stat from the overlay indicated by stat_name. It is the same name as was used in add_stat.
