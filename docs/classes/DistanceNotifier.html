DistanceNotifier

Measures the distances between two things

Inherits from Node2D

Description

This node measures the distance between itself and the node specified by the other_node property.

What exactly gets measured depends on the component property. Either the X or Y distance between two nodes can be checked, or both. Alternatively, the linear distance between the two nodes can be checked.

Default NodePath()

• int component — This determines what gets measures exactly.

Default X_COMPONENT

Valid values are: X_COMPONENT, Y_COMPONENT, BOTH_COMPONENTS, or FLOAT. X_COMPONENT and Y_COMPONENT are obvious, but BOTH_COMPONENTS and FLOAT need explaining.

With BOTH_COMPONENTS, both the X and Y component of threshold are checked: the X and Y distance between two nodes both need to be less than or equal to the threshold's X and Y component, respectively, in order to be considered "within range".

With FLOAT, the linear distance between the two nodes is calculated. Imagine a line between the connected nodes: the length of that line is the distance being checked under this mode. In this case, the X component of threshold is used as the threshold.

• Vector2 threshold — Maximum distance between nodes

Default Vector2()
Setter set_threshold(value)

What you put in here depends on component: if X_COMPONENT or Y_COMPONENT, either the X component or the Y component are used; if BOTH_COMPONENTS, both components are used; if FLOAT, the X component is used to hold a linear distance.

In the case of FLOAT, the linear distance refers to the result of Vector2::distance.

Constants

  • X_COMPONENT = 0 -- Measure only the X distance
  • Y_COMPONENT = 1 -- Measure only the Y distance
  • BOTH_COMPONENTS = 2 -- Measure both the X and Y distances
  • FLOAT = 3 -- Calculate the linear difference between two nodes

Signals

• entered_range(Node2D node)

The other node is closer than threshold.

• exited_range(Node2D node)

The other node is farther away then threshold.

Methods

• Vector2 get_vector()

Returns the distance between this and the other node.

This function checks the distance between this node and the one referenced by other_node. The return valued is a Vector2 specifying the positive distance between the two.

If other_node is empty or points to an invalid node, Vector2() is returned and an error is emitted.

• bool is_within_range()

Returns true if the other node is within range.

• void set_other_node(Variant v)

Set the other_node property.

Sets the node to be measured by DistanceNotifier. If v is a Node2D-derived type, the path is derived from its position in the tree.
